# PwnDatas-DB-Project
PwnDatas-DB-Project(PDDP)

安裝依賴：
```
pip3 install pymediawiki
```

使用：
```
cd /opt
git https://github.com/JustYoomoon/PwnDatas-DB-Project.git
cd

.bashrc加入
alias vulsearch="python3 /opt/PwnDatas-DB-Project/vulsearch.py"

source .bashrc
```

```
vulsearch <NAME>

舉例：
vulsearch CVE-2019-0708
```
